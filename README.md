### Installation

```bash
npm install @toojs/stylelint-config-standard --save-dev
// or
yarn add @toojs/stylelint-config-standard -D
```

### Extending the config

```json
{
  "extends": " @toojs/stylelint-config-standard",
  "rules": {
    // your rules
  }
}
```
